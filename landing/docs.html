Myca'ya Hoş Geldiniz

Myca, bilgisayarlarınızı otonom bir yapay zeka ağı haline getiren yerel öncelikli (local-first) bir yapay zeka otomasyon işletim sistemidir.

5 Dakikada Hızlı Kurulum

Myca'yı kurmak ve ilk yerel otomasyon akışınızı başlatmak için bu basit adımları izleyin.

# 1. Depoyu klonlayın ve dizine gidin
git clone https://github.com/brienteth/myc-ai
cd myc-ai/ai-layer

# 2. Gereksinimleri kurun ve backend'i başlatın
pip install -r requirements.txt
python main.py

# 3. Masaüstü uygulamasını başlatın
cd ../desktop
npm install
npm run electron:start

İlk Başarı: Downloads Klasörünü Özetle

Arayüze gidin ve AI Assist çubuğuna şu komutu yazın:

"Summarize my Downloads folder."

Myca otomatik olarak bir Plan (DAG) üretecek, indirilenler klasöründeki dosyaları okuyacak, yerel modeliniz yardımıyla özetleyecek ve masaüstünüze bir rapor olarak kaydedecektir.

Niyet Nasıl Yorumlanır?

Gelen niyet yerel semantik önbellek filtrelerinden geçirilir. Eğer daha önce benzer bir işlem yapılmışsa, öğrenilen deneyimlerden (Experience) doğrudan rota belirlenir. Aksi halde yerel LLM planlama yapması için devreye sokulur.

Planner Execution Graph

Planlayıcı (Planner)

Planlayıcı, niyet ifadesini analiz eder ve kullanılabilir becerileri (Skills) kullanarak sıralı/paralel adımlardan oluşan deterministik bir plan hazırlar.

# planner.py içindeki örnek girdi/çıktı akışı
Input: "Faturayı oku ve Excel'e yaz"
Output: ExecutionGraph JSON [fs.read ➔ ai.summary ➔ fs.write]

Yapay Zeka Destekli Planlama ve Hızlı Yol (Bypass)

Myca, yaygın niyetler için önceden eğitilmiş hızlı şablonlar (Heuristic bypass) sunarak yerel CPU/GPU yükünü en aza indirir. E-posta, dosya işlemleri, sosyal medya paylaşımları anında deterministik birer grafiğe dönüştürülür.

İlişkili Konseptler

Execution Graph Skills

Yürütme Grafiği (Execution Graph)

Yürütme Grafiği, bağımlılıkları belirlenmiş düğümlerden (Nodes) oluşan Yönlendirilmiş Döngüsüz bir Grafiktir (DAG). Adımlar paralel veya sırayla çalıştırılabilir.

[Düğüm A: fs.read]
↓
[Düğüm B: ai.summary] ➔ (Veriyi İşle)
↓
[Düğüm C: email.send] & [Düğüm D: telegram.send] (Eşzamanlı)

Hata Yönetimi ve İptal Mekanizması

Düğümler çalışırken her adımda durum izlemesi yapılır. Eğer bir düğüm çökerse, akış iptal edilir veya kurtarma (retry) mekanizmaları devreye girer. Kullanıcı dilerse arayüzdeki "Stop" butonu ile yürütmeyi anında sonlandırabilir.

Guides Skills

Geliştirici SDK

Kendi becerilerinizi (Skills) yazmak ve Myca OS ile entegre etmek için Python SDK'mızı kullanabilirsiniz.

Skill Tanımlama Örneği

Her beceri girdi parametreleri, çıktılar ve gerekli izin bildirimleri ile tanımlanır.

from myca.skills.core.decorator import skill

@skill(
    id="custom.file_printer",
    version="1.0.0",
    permissions=["fs.write"],
    inputs=["file_path", "text"],
    outputs=["status"]
)
async def print_file(ctx, file_path: str, text: str) -> dict:
    ctx.emit("custom.file_printer.started", {"file_path": file_path})
    await ctx.check_cancel()
    
    with open(file_path, "w") as f:
        f.write(text)
        
    return {"status": "Success"}

API Referansı Mimari

REST API Referansı

Myca OS yerel localhost `8420` portu üzerinde çalışır. Uygulamayı dışarıdan kontrol etmek için aşağıdaki endpointleri çağırabilirsiniz.

Endpoints

POST /query

İş akışlarını veya yapay zekayı tetiklemek için ana endpoint.

Request:
{
  "prompt": "Summarize this downloads folder",
  "stream": false
}

Response:
{
  "response": "Summary data...",
  "done": true
}

GET /automation/history

Daha önce çalıştırılmış akışların geçmiş durumlarını döndürür.

POST /automation/cancel

Aktif yürütülmekte olan akışları anında durdurur.

Sıkça Sorulan Sorular

Myca verilerimi internete yükler mi?

Hayır. Myca yerel öncelikli çalışır. Cihazlarınız arasındaki iletişim yerel ağda WebRTC üzerinden doğrudan (P2P) şifreli olarak gerçekleşir.

İnternet bağlantısı olmadan otomasyonlar çalışır mı?

Evet, yerel bilgisayarınızdaki klasör izlemeleri, Excel yazımları ve yerel yapay zeka analizleri tamamen çevrimdışı çalışabilir. Sadece e-posta gönderme veya sosyal medya yüklemesi gibi işlemler internet gerektirir.

Yol Haritası (Roadmap)

  • Q3 2026: Gelişmiş P2P model shard bölme ve dağıtık çıkarım desteği.
  • Q4 2026: Mobil kütüphane (iOS/Android) ve yerel sesli komut asistanı.
